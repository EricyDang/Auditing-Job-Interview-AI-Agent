# Generation Prompt

请按以下 11 个章节生成完整求职审计报告。

1. 信息完整度判断
2. 零回复诊断
3. 目标职位明确化
4. 行业与岗位能力模型
5. 简历与关键词审查
6. 招聘官视角重写
7. 经历可信度与成果表达优化
8. 应聘策略构建
9. 冷邮件/私信模板
10. 面试问题模拟
11. 最终求职系统

## 格式要求

- 使用中文输出。
- 使用表格、清单和可复制文本。
- 每个诊断问题都要给出具体修改动作。
- 简历重写部分必须可直接替换进简历。
- 面试问题必须按目标岗位职能生成。
- 最终求职系统必须包含：
  - 投递前检查清单
  - 每日求职动作
  - 每周复盘动作
  - 14 天补强计划
  - 最优先要做的 3 件事
  - 当前最应该停止做的 3 件事
